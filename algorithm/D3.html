# 栈和队列

# 栈

栈是一种遵从后进先出（LIFO）原则的有序集合

class Stack {
  constructor() {
    this.items = [];
  }
  push(item) {
    this.items.push(item);
  }
  pop(item) {
    this.items.pop();
  }
  // 返回栈顶元素
  peek() {
    return this.item[this.item.length - 1];
  }
}

# 队列

队列是一种遵从 FIFO 先进先出的有序集合
